# 🎨 UI/UX Wireframes

## 1. Interaction Model
DNA is **Voice-First**. It does not have a traditional GUI window. Interaction is handled via the System Tray and Toast Notifications.

## 2. System Tray Icon States
- **⚪ Grey/Dim:** Idle / Off.
- **🔵 Pulsing Blue:** Listening for speech.
- **🟡 Spinning Yellow:** Thinking / Processing.
- **🟢 Solid Green:** Speaking.
- **🔴 Red Pulse:** Error / Restricted Action.

## 3. Notifications (Windows Toast)
- **Status Updates:** "Thinking...", "Found your file...", "Running script..."
- **Feedback:** Success/Failure messages.
- **Interactive:** Quick buttons for "Confirm" or "Cancel" on high-risk actions.

## 4. Audio Feedback
- **Chimes:** Distinct start/stop listening tones.
- **Errors:** Subtle low-frequency buzz for failures.
